    gate.metrics.core.accuracy_top_k(logits: Tensor, labels: Tensor, k: int) → Tensor[source]

    Computes the top-k accuracy for a given set of logits and labels.

    Parameters:

    • logits -- A tensor of shape (batch_size, num_classes) containing the predicted logits for each example in the batch.

    • labels -- A tensor of shape (batch_size,) containing the ground truth labels for each example in the batch.

    • k -- The value of k for the top-k accuracy calculation.

    Returns:

    A tensor of shape (k,) containing the top-k accuracy for each value of k.
